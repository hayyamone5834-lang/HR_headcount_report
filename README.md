# HR_headcount_report
An interactive Power BI dashboard that analyzes total headcount, workforce demographics, departmental distribution, and geographic trends. Designed with DAX measures and a clean data model to deliver clear, actionable HR insights for decision-makers.

# Project Overview

This project features a comprehensive HR Analytics Dashboard built in Power BI to provide stakeholders with actionable insights into workforce dynamics. The dashboard transforms raw HR data into interactive visualizations to support data-driven decision-making regarding recruitment, retention, and employee performance.

# Key Features

Headcount Tracking: Real-time monitoring of total employees across different departments and locations.
Attrition Analysis: Identification of turnover trends and key drivers behind employee departures.
Demographic Insights: Breakdown of the workforce by age, gender, and tenure.
Performance Metrics: Visualization of employee appraisal scores and productivity trends.
Interactive Slicers: Filter data by Year, Department, Region, and Employment Type for granular analysis.

# Tools & Technologies

Power BI Desktop: Used for data modeling, DAX calculations, and report visualization.
Power Query: Employed for ETL (Extract, Transform, Load) processes and data cleaning.
DAX (Data Analysis Expressions): Utilized to create complex measures like Year-over-Year (YoY) Growth and Attrition Rate.
Data Source: CSV
